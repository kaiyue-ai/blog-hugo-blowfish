欢迎来到这个新的个人博客。它使用 Hugo 生成静态页面，使用 Blowfish 作为主题，并通过 Vercel 免费部署。

## 为什么选择静态博客

静态博客没有数据库和后端服务，维护成本很低。你只需要关注内容本身：写 Markdown、提交到 GitHub，然后让 Vercel 自动构建发布。

## 代码块示例

```go
package main

import "fmt"

func main() {
  fmt.Println("Hello, Hugo!")
}
```

你可以把这里改成日常记录、技术笔记、读书摘要或项目复盘。分类适合放大主题，标签适合放更细的关键词。
